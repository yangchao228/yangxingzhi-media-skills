# 小红书爆款图文图片集合生成器 v4

这是实战增强版 skill，解决：

**“我有一篇长文，想一键转成小红书爆款图文，而且最好连发布文案都一起给我。”**

## v4 新增能力

### 1）自动判断 7 / 8 / 9 张
它不会再机械默认 9 张，而是会根据文章内容自动判断：
- 7 张是否更精炼
- 8 张是否更平衡
- 9 张是否更完整

### 2）直接输出可发小红书的文案包
新增自动输出：
- 3 个标题候选
- 主发布配文
- 简短口语版配文
- 话题标签
- 评论区引导文案
- 私信 / 收藏引导文案

### 3）保留 v3 能力
- 多风格切换
- 风格自动推荐
- 文件直读
- 自动拆卡
- 统一视觉规范
- 逐页出图提示词
- 如果环境支持则直接出图

## 支持风格
- 贴纸风（默认）
- 极简轻科技风
- 深色专业风
- 卡通风

## 推荐怎么用

### 方式一：默认自动判断
```text
请使用 xiaohongshu-viral-image-set-generator-v4，把下面这篇长文转成小红书图文，并自动判断最适合做 7 张、8 张还是 9 张。
```

### 方式二：加上风格要求
```text
请使用 xiaohongshu-viral-image-set-generator-v4，把这篇文章转成小红书图文，风格使用贴纸风，并附上可直接发布的小红书文案包。
```

### 方式三：读取文件并出图
```text
请使用 xiaohongshu-viral-image-set-generator-v4，直接读取我上传的 markdown 文件，自动判断页数并生成小红书图文。如果环境支持图片生成，请继续出图。
```
