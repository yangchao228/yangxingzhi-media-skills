# quick-start.md

## 1. 最简单用法
把上一层 AI / Human3.0 选题 skill 的 `final_pick` 和作者真实素材一起填入输入模板。

## 2. 一句话触发
- 用 AI/Human3.0 版公众号写作 skill 写这篇
- 按 Human3.0 栏目风格把这个题写成公众号稿

## 3. 推荐上游输入
- final_pick.title
- final_pick.reason
- article_type
- top3
- author_context
- column_direction

## 4. 最适合的场景
- AI 热点转成长文
- Agent 实践复盘
- Human3.0 专栏写作
- AI 养娃案例文
- 工程师 AI 提效文章
