# AI / Human3.0 专用微信公众号热点选题 Skill

这是一个**增强版**的微信公众号热点选题技能包，专门服务这些主题：
- AI / Agent / OpenClaw
- Human3.0 / 认知主权 / 个体升级
- 工程实践 / 开发者转型 / AI 工作流
- 家庭协作官 / AI 真实落地场景

它不是“AI 新闻总结器”，而是把热点转成更有深度的公众号选题：
1. 从产品更新里看到结构变化；
2. 从行业新闻里提炼对普通人的意义；
3. 从热点讨论里转出可以写成长文的观点和方法；
4. 尽量接到作者自己的实践与案例。

## 这个版本的重点
- 强化“结构变化、门槛变化、组织方式变化、工作流变化”判断
- 强调 Human3.0 / Agent / 真实落地场景的连接
- 更警惕资讯搬运感
- 更适合做你自己的专栏型账号

## 目录说明
- `SKILL.md`：增强版技能说明
- `SYSTEM_PROMPT.md`：系统级 prompt
- `templates/`：输入输出模板
- `output-schema.json`：结构化 schema
- `scorecard.md`：AI 向选题评分卡
- `angle-library.md`：AI/Human3.0 常用切角库
- `examples/`：few-shot 示例
- `quick-start.md`：快速开始

## 推荐工作流
AI 新闻 / Agent 热点 / 你的实操素材 → 本 skill 做切题与排序 → 选出 Top 1 → 交给公众号写作 skill
