# AI / Human3.0 选题评分卡

每个选题按 6 个维度打分，每项 1-10 分，总分 60 分。

## 1. 主题相关度
是否和 AI / Agent / Human3.0 主线高度一致。

## 2. 结构变化强度
是否体现出能力门槛、工作流、组织方式、个体生产方式的变化。

## 3. 公众号适配度
是否适合写成长文，而不是一则快讯。

## 4. 获得感
是否能给出趋势判断、框架、做法、场景化启发。

## 5. 真人感
是否能接到作者真实项目、实践、踩坑、案例。

## 6. 品牌沉淀价值
是否有利于沉淀 Human3.0 / Agent / 实践专栏，而不是一次性消费。

## 建议阈值
- 48-60 分：优先写，可直接进入创作
- 38-47 分：可写，但需优化切角
- 28-37 分：一般，不建议优先写
- 28 分以下：建议放弃
