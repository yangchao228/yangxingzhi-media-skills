# 示例：OpenClaw / Agent 方向

## 输入
账号主题：OpenClaw / AI Agent
账号定位：个人实践 + 利他干货 + 趋势观察
目标读者：想学 AI 的普通上班族、想提效的个体创作者
内容目标：涨粉 + 建立人设 + 为后续 skills 引流
作者近期经历：用 OpenClaw 做微信助手、家庭协作官、RSS 信息收集系统
热点素材：近期 AI Agent 产品更新、模型厂商发布会、关于 Agent 的热门讨论
额外限制：不要纯资讯复述，要有真人感、获得感、公众号长文价值
长期栏目：OpenClaw 养虾日记 / AI 助手落地实践

## 期望输出方向
- 不要只总结“某家又发了个 agent”
- 优先输出“普通人为什么现在该理解 Agent”
- 优先输出“从会聊天到能干活，这中间差了什么”
- 尽量接入作者自己的 OpenClaw 实操案例
