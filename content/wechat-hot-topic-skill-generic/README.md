# 通用型微信公众号热点选题 Skill

这是一个**通用版**的微信公众号热点选题技能包，适合几乎所有“有明确主题主线”的公众号账号。

它的目标不是抓热点，而是回答 4 个关键问题：
1. 这个热点和我的主题到底有没有关系？
2. 这个热点适不适合写公众号长文？
3. 它更适合从什么角度切进去？
4. 今天最值得先写哪一篇？

## 适用主题
- 个人成长 / 职场 / 教育
- 科技 / 产品 / 商业 / 消费趋势
- 家庭 / 育儿 / 生活方式
- 行业观察 / 经验复盘 / 个人品牌

## 核心特点
- 不硬蹭热点
- 更强调公众号长文适配
- 输出带判断逻辑，不只给标题
- 兼顾热点借势和长期栏目沉淀
- 适合接任意写作 skill 或内容工作流

## 目录说明
- `SKILL.md`：技能主说明
- `SYSTEM_PROMPT.md`：系统级 prompt
- `templates/input-template.md`：输入模板
- `templates/output-template.md`：输出模板
- `output-schema.json`：结构化输出 schema
- `scorecard.md`：选题评分卡
- `examples/`：通用场景 few-shot 示例
- `quick-start.md`：快速接入说明

## 推荐工作流
热点素材 → 本 skill 做筛选与排序 → 选出 Top 1 / Top 3 → 交给写作 skill

## 这个版本最适合什么人
适合你后面做成“底座技能”：
同一套逻辑可以服务多个主题账号，只要换输入信息就能复用。
