## 正文快照

文章主线：Token 成本不是单纯字数账，它会暴露一个人如何组织上下文、工具调用、模型选择和工作流复盘。

核心段落摘要：

- AI 账单变贵，很多时候不是问得多，而是每一问都太重。
- 输入 Token 包括提示词、系统消息、项目说明、文件内容、对话历史和工具返回结果。
- 输出、缓存、推理 Token 分别对应回答生成、稳定前缀复用和复杂任务中的思考成本。
- 开发者高频浪费来自整仓喂给 AI、工具循环失控、简单任务用高端模型、缓存前缀不稳定和长对话无限滚大。
- 真正的降本动作是先搜索、再提问，先规划、再读文件，把稳定规则、项目说明和高频任务沉淀成可复用资产。

最适合入书的一句话：

> Token 是账单，流程才是资产。
