# Human3.0 Book Archive

这个目录用于维护《Human3.0》成书素材库。

它不是发布目录，也不是所有公众号文章的备份目录。只有经过 `human3-book-guardian` 审查，并且用户确认进入成书素材库的内容，才进入这里。

## 目录结构

```text
human3.0_book/
  README.md
  materials.md
  entries/
    YYYY-MM-DD-slug.md
```

- `materials.md`：成书素材总索引，按 Part 1 到 Part 6 归类。
- `entries/`：单篇素材卡片，保存审查结论、归档判断、发布包和正文快照。

## 维护规则

1. 每篇文章先完成公众号出刊检查，再进入成书审查。
2. 只有用户明确确认“进入 Human3.0 归档”后，才新增 entry。
3. 每个 entry 至少包含：
   - 原始标题
   - 书稿备注标题
   - 审查结论
   - 归属主线
   - 建议章节
   - 最小纠偏
   - 状态标签
   - 正文快照或正文文件引用
4. `materials.md` 只放索引和一句话判断，不承载全文。
5. 如果一篇文章后续重写，不覆盖旧 entry；新增修订记录或新 entry。
